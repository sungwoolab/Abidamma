## 모듈별 상세 내용

### 1. index.html — 홈
- 상단 헤더 + 히어로 섹션
- 5개 학습 모듈 카드 (학습 가능 / 잠금 상태 표시)
- 라이트 테마 (배경 #f8f7f4, 보라색 포인트 #6b46c1)
- 모든 모듈 링크 연결 완료

### 2. intro.html — 아비담마란 무엇인가?
- 브레드크럼 + 진도 바
- 학습 단계: 개요 → 역사 → 핵심 특징 → 퀴즈
- 퀴즈 (선택형, 즉각 피드백)
- 완료 배너 → 홈으로 이동

### 3. four-realities.html — 4법 개요
- 4법 카드: 마음(Citta) / 마음부수(Cetasika) / 물질(Rūpa) / 열반(Nibbāna)
- 카드 클릭 시 상세 설명 팝업
- 서브탭: 마음 89종 / 마음부수 52종 / 물질 28종 / 열반
- 각 탭별 퀴즈 포함

### 4. vithi.html — 인식과정
- **서브탭 1: 17심찰나**
  - 각 찰나(과거·문의·등록·결정·속행7·등록2) 클릭 인터랙션
  - 클릭 시 해당 찰나의 마음부수 표시
- **서브탭 2: 선정 인식과정**
  - 색계 5선정 단계별 선정지 변화 애니메이션
  - 무색계 4선 (공무변처·식무변처·무소유처·비상비비상처)
- **서브탭 3: 도·과 인식과정**
  - 수다원·사다함·아나함·아라한 4단계

### 5. life-rebirth.html — 삶과재생
- **서브탭 1: 한 생의 마음 흐름**
  - 재생연결심 → 바왕가 → 인식과정들 → 죽음심 타임라인
- **서브탭 2: 재생연결 인식과정**
  - 5단계 흐름도 (죽음 직전 대상 → 재생연결심)
- **서브탭 3: 31 존재계**
  - 클릭 펼침 지도 (악처·인간계·욕계천·색계·무색계)

### 6. advanced.html — 심화·응용
- **서브탭 1: 조건 관계 Paṭṭhāna**
  - 24조건 전체 카드 클릭 탐색
- **서브탭 2: 수행과의 연결**
  - 위빠사나 수행에서 아비담마 개념 적용
  - 5단계 적용 가이드 + 케이스 스터디
